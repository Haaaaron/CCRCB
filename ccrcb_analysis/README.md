# CCRCB Data Analysis Package

This package provides parsing and plotting utilities for CCRCB benchmark results. It processes timing data (`results.txt`) and hardware metrics (`.sqlite` files generated by Nsight Systems).

## Structure
- `parsers.py`: Contains functions to read benchmark outputs and return Pandas DataFrames.
- `plotters.py`: Contains Matplotlib-based functions for generating multi-dimensional visualizations.

## Usage

You can use this package directly in Jupyter notebooks. All plots accept Matplotlib `**kwargs` for customization.

```python
import ccrcb_analysis.parsers as parsers
import ccrcb_analysis.plotters as plotters

# Load data
df = parsers.load_run_data("results_profile_...")

# Plotting
import matplotlib.pyplot as plt

fig, ax = plt.subplots(figsize=(8, 6))
plotters.plot_scaling_lines(df, x_metric="Math_Load", y_metric="Serial_Speedup", group_by="Backend", ax=ax, title="Speedup vs Math Load")
plt.show()
```
